=============
 hygrometry
============

Helpful hygrometry python functions, like dew point, wetbulb, absolute humidity, mixing ratio, etc.

For install instructions and API see the documentation `here<https://bluthen.github.io/hygrometry_docs/>`_.

A jupyter notebook with some simple analysis on the wetbulb algorithm `here<https://nbviewer.jupyter.org/github/bluthen/hygrometry/blob/master/docs/hygrometry.ipynb>`_.
